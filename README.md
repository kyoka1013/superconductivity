# 超伝導実験 解析データ

## 概要
本リポジトリは、銅酸化物系高温超伝導体 YBa₂Cu₃O₇−δ の電気抵抗測定実験で得られたデータを解析し、
超伝導転移温度 \(T_c\) を決定するための解析コードおよび関連データをまとめたものである。

実験では四端子法を用いて電気抵抗の温度依存性を測定し、
取得した生データに対して電圧スケール変換、外れ値処理、R–T 曲線の作成を行った。
解析は Jupyter Notebook 上で再現可能な形で実装されている。

---

## 解析手法
解析では以下の手順を採用している。

1. レコーダにより取得された電圧・温度データの読み込み  
2. 電圧レンジ切替に対応した実電圧への換算  
3. 定電流値を用いた抵抗値への変換（四端子法）  
4. 降温過程データの選定および外れ値の除去  
5. 抵抗–温度（R–T）曲線の作成  
6. R–T 曲線に基づく超伝導転移温度の算出  

これらの処理は Notebook 上で一貫して行われており、
解析条件を変更することで結果の再計算が可能である。

---

## 転移温度解析（Jupyter Notebook）
超伝導転移温度 \(T_c\) は、high-Tc 実験テキストに基づき、
以下の 3 種類の定義を用いて評価している。

- \T_{c,\mathrm{onset}}\：抵抗が常伝導状態から減少し始める温度  
- \(T_{c,\mathrm{mid}}\)：転移領域における抵抗の中間値をとる温度  
- \(T_{c,\mathrm{zero}}\)：抵抗が測定限界以下となる温度  

Jupyter Notebook では、これらの温度を R–T 曲線から数値的に抽出し、
同一手法を用いた再解析が可能となっている。
